Create a proxy app for the tethys portal

The following form can be used to create a Tethys proxy app. Proxy apps are links to external web applications or websites. They are represented in the App Library page with an app tile just like native Tethys Apps.

After submitting the form, the page will reload automatically to add the new proxy app to the tables

A name of the proxy app must be provided

A endpoint for the proxy app must be provided
